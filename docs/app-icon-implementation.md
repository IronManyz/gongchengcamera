## 🔧 技术实现详情

### 1. 应用图标配置

#### 配置文件: `AppScope/app.json5`
```json5
{
  "app": {
    "bundleName": "com.example.engineeringcamera",
    "icon": "$media:layered_image",
    "label": "$string:app_name",
    "versionCode": 1000100,
    "versionName": "1.0.1"
  }
}
```

#### 图标资源路径
- **主图标**: `entry/src/main/resources/base/media/layered_image.png`
- **图标格式**: PNG，支持透明通道
